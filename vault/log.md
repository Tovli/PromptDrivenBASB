# Vault Knowledge Log

Append-only human-readable log of meaningful knowledge events. This file is a package-owned operational artifact, not a P.A.R.A. category. Write here when any of the following happens:

- a durable source is ingested into `vault/sources/`
- one or more compiled notes receive a substantive synthesis update
- a contradiction between compiled notes is resolved
- a reusable expressed output is promoted back into the vault
- a major note merge, split, or refactor lands

Keep entries short: one paragraph or a few bullets. Link to the compiled notes, source notes, and prompt that drove the change.
